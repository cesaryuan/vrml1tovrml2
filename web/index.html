Suggested GitHub Description

Repository tagline

A Rust-powered VRML 1.0 to VRML 2.0 converter for Linux, WSL, and CI workflows, with browser-based conversion via GitHub Pages.

How It Works

Online usage

  1. Upload a legacy VRML 1.0 .wrl file or load the sample scene.
  2. Click Convert in Browser to run the Rust converter in WebAssembly.
  3. Review the VRML 2.0 result, then copy or download it for downstream use.

Files stay in your browser session and are not uploaded to a server. 所有转换都在本地浏览器内完成。

CLI Workflow

Batch conversion on your machine

./vrml1tovrml2 input_v1.wrl output_v2.wrl
./vrml1tovrml2 --progress input_v1.wrl output_v2.wrl
cargo test --test public_v1_regression

The browser tool is ideal for quick checks. For batch jobs and regression testing, keep using the repository CLI.
