VRML Converter Demo

vrml1tovrml2

浏览器内将 VRML 1.0 .wrl 转成 VRML 2.0。适合快速验证格式，批量处理仍建议使用仓库里的 CLI。

查看 GitHub 项目

Rust + WASM Local Only

在线转换

上传文件或粘贴文本后点击转换，结果会直接显示在右侧。

Initializing WASM...

The page is loading the WebAssembly converter. 页面正在加载浏览器内转换器。

使用说明

  1. 选择本地 `.wrl` 文件，或者先加载示例。
  2. 点击“开始转换”，在浏览器内运行 Rust WebAssembly 转换器。
  3. 检查右侧输出，再复制或下载结果。

页面不上传文件，所有处理都在当前浏览器会话中完成。

CLI 用法

./vrml1tovrml2 input_v1.wrl output_v2.wrl
./vrml1tovrml2 --progress input_v1.wrl output_v2.wrl
